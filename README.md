# bash-scripts
# 🛠️ bash-scripts – Bộ sưu tập script cài đặt tự động

Chào bạn! Đây là kho chứa các **Bash script đơn giản mà hiệu quả** giúp tự động hoá việc cài đặt phần mềm, cấu hình hệ thống, tối ưu máy chủ Linux... dành cho những người:

- 💻 Thường xuyên cài lại server
- 🚀 Muốn tiết kiệm thời gian gõ lệnh thủ công
- 🤓 Yêu Bash, thích CLI

---

## 📦 Nội dung repo

- `install-nginx.sh` – Cài đặt và cấu hình Nginx chuẩn cho production
- `setup-docker.sh` – Cài đặt Docker + Docker Compose
- `install-nodejs.sh` – Cài Node.js mới nhất theo cách an toàn
- `secure-ubuntu.sh` – Tối ưu và bảo mật Ubuntu Server (fail2ban, ufw, v.v.)
- *(và nhiều script khác đang viết dần)*
